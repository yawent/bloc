open terminal, type 
```
mkdir checkpoint_exercise
```
then a directory named checkpoint_exercise has been created. Type
```
cd /users/tian/checkpoint_exercise
```
to navigate into checkpoint_exercise. Type
```
touch my_cl_checkpoint.txt
```
then a file named my_cl_checkpoint has been created in the checkpoint_exercise directory.
Type 
```
open my_cl_checkpoint.txt
```
then the file has been opend, and type "Hello World!" in the txt.
Type
```
cat my_cl_checkpoint.txt
```
then the contents "Hello World!" is printed in my shell.
Type
```
mv my_cl_checkpoint.txt cl_checkpoint.txt
```
then the file is renamed.
Type 
```
cat cl_checkpoint.txt
```
then the contents of the renamed file is printed in my shell.

```
history
```
